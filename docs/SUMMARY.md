# 项目总览

## 1. 项目目的

本项目研究常规单主旋翼带尾桨直升机在涡环状态（VRS）下的建模与仿真。当前目标是把阶段一资料整理为阶段二模型规格，使后续实现能够围绕同一套坐标系、状态量、模块接口和验证场景推进。

阶段二的总模型可先抽象为：

$$
\dot{x}=f(x,u,t)
$$

其中，$x$ 是直升机状态向量，$u$ 是操纵输入，$t$ 是时间。坐标方向、姿态角、速度分量和力矩正方向已按 [轴系与符号冻结约定](axis-conventions.md) 统一。

## 3. 建模范围

当前建议的建模对象是常规单主旋翼带尾桨直升机。模型范围包括：

- 机体刚体六自由度运动。
- 主旋翼气动力和力矩。
- 尾桨侧向力和反扭矩平衡。
- 机身、平尾、垂尾的简化气动力。
- 旋翼诱导速度。
- VRS 判别与修正。
- 典型进入和改出场景。

VRS 应作为旋翼入流和有效拉力的异常状态进入模型：

```text
飞行状态 + 主旋翼状态
  -> VRS_Detection
  -> VRS_Correction
  -> 修正诱导速度 / 修正有效拉力 / 加入扰动
  -> 主旋翼力和力矩
```

## 6. 验证主线

验证应先覆盖五类场景：

1. 悬停：旋翼拉力接近重量，垂向加速度接近零。
2. 正常下降：不触发 VRS，垂向速度响应稳定。
3. 低速大下降率：触发 VRS 标志，诱导速度异常，等效拉力下降或垂向速度增大。
4. 改出：VRS 标志退出，垂向速度恢复。
5. 参数扫掠：得到随水平速度、下降率和总距变化的 VRS 边界。

更详细的验证拆分见 [验证路线图](verification-roadmap.md)。
