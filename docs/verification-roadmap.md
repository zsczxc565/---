# 验证路线图

> 状态：草案。本文定义阶段二模型的验证顺序、观测信号和数据缺口，不替代后续测试脚本。

## 1. 验证原则

- 先验证正常飞行和配平，再验证 VRS 进入和改出。
- 先验证量级和趋势，再追求高保真数据拟合。
- 每个 VRS 结果都需要和非 VRS 反例对照，避免判据过宽。
- 所有验证信号必须能追溯到模型变量，例如 $V_h$、$V_d$、$v_i$、$T_\mathrm{eff}$ 和 $s_\mathrm{VRS}$。

## 2. 验证场景矩阵

| 场景 | 输入条件 | 应观察信号 | 主要依据 |
|---|---|---|---|
| 悬停 | 水平速度接近零，下降率接近零，总距配平 | $T \approx W$，垂向加速度接近零，$s_\mathrm{VRS}=0$ | 课程第 5 章、建模依据 |
| 正常下降 | 中等下降率，非低速大功率危险组合 | 不触发 VRS，垂向速度稳定 | FAA VRS 判据反例 |
| 低速大下降率 | 低水平速度，有明显下降率，有功率输入 | $s_\mathrm{VRS}>0$，$v_i$ 异常，$T_\mathrm{eff}$ 下降或垂向速度增大 | FAA + Johnson |
| 改出 | 前推周期变距或调整总距后恢复 | $s_\mathrm{VRS}$ 退出，垂向速度恢复 | FAA 改出逻辑、飞行动力学常识 |
| 参数扫掠 | 改变水平速度、下降率和总距 | 得到 VRS 边界图或危险区表 | Johnson VRS 边界思想 |

## 3. 信号清单

| 信号 | 含义 | 验证用途 |
|---|---|---|
| $V_h$ | 水平速度量级 | 判断是否低于危险低速范围 |
| $V_d$ | 下降率 | 判断是否进入大下降率区域 |
| $\theta_0$ | 主旋翼总距 | 表示功率/拉力输入状态 |
| $v_i$ | 主旋翼平均诱导速度 | 观察正常入流与 VRS 修正差异 |
| $T_\mathrm{nom}$ | 名义旋翼拉力 | 作为修正前基准 |
| $T_\mathrm{eff}$ | VRS 修正后的有效拉力 | 观察拉力损失 |
| $s_\mathrm{VRS}$ | VRS 标志或强度 | 进入、维持和退出判据 |
| $\dot{H}$ | 高度变化率 | 观察下沉和改出响应 |

## 4. 阶段验证顺序

### 4.1 规格级检查

- 检查坐标系、正方向和单位是否已冻结。
- 检查状态向量 $x$ 和输入向量 $u$ 是否与实现代码一致。
- 检查 `VRS_Detection` 和 `VRS_Correction` 的输入输出是否完整。

### 4.2 单模块检查

- `InflowModel`：悬停时诱导速度量级合理。
- `MainRotor`：总距增大时名义拉力 $T_\mathrm{nom}$ 增大。
- `VRS_Detection`：正常下降不触发，低速大下降率触发。
- `VRS_Correction`：$s_\mathrm{VRS}$ 增大时修正连续，不出现非物理跳变。

### 4.3 集成场景检查

- 悬停场景：验证 $T \approx W$。
- 正常下降场景：验证 $s_\mathrm{VRS}=0$。
- VRS 进入场景：验证 $s_\mathrm{VRS}>0$ 且垂向响应变差。
- 改出场景：验证 $s_\mathrm{VRS}$ 下降至安全区域。
- 参数扫掠：验证危险区边界随 $V_h$、$V_d$ 和 $\theta_0$ 合理变化。

## 6. 验收门槛

阶段二第一版模型通过验证的最低门槛：

- 悬停时 $T/W$ 接近 $1$，且垂向加速度量级合理。
- 正常下降不会错误触发 VRS。
- 低速大下降率会触发 VRS。
- VRS 触发后能观察到 $T_\mathrm{eff}<T_\mathrm{nom}$ 或下沉趋势增强。
- 改出输入后 $s_\mathrm{VRS}$ 能回落。
- 参数扫掠能输出可解释的危险区边界。
