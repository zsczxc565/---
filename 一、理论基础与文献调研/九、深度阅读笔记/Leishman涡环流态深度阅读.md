# Leishman VRS 流态深度阅读笔记

来源：J. Gordon Leishman, *Principles of Helicopter Aerodynamics*，第 2 章，轴向爬升与下降动量分析附近；PDF 页约 128-129，对应书页约 90-91。

## 1. 一句话摘要

Leishman 把轴向下降中的旋翼流态分成正常工作、涡环、湍流尾迹、风车制动四类，说明 VRS 不是普通下降的连续小扰动，而是旋翼尾迹拓扑和非定常载荷发生质变的区域。

## 2. 结构地图

```text
轴向下降速度逐渐增大
  -> 正常工作状态
    -> 尾迹仍近似规则
  -> 涡环状态
    -> 桨尖涡靠近桨盘
    -> 旋翼平面附近形成涡环
    -> 载荷非定常、控制效能下降
  -> 湍流尾迹状态
    -> 尾迹高度混乱
    -> 可见强非周期流动
  -> 风车制动状态
    -> 旋翼从气流中取能
    -> 对应自转/风车制动分析
```

## 3. 对本项目的关键启发

### 3.1 VRS 应建成“区域状态”，不是单个阈值

VRS 前后存在多个流态。如果模型只有一个二值开关：

$$
\mathrm{VRSFlag}\in\{0,1\}
$$

它会过于粗糙。更合理的是先设计成分段状态：

$$
s_{\mathrm{rotor}}\in
\{\mathrm{normal},\mathrm{vortex\ ring},\mathrm{turbulent\ wake},\mathrm{windmill\ brake}\}
$$

第一版 Simulink 可以仍然只输出 `VRSFlag`，但内部最好保留 `RotorFlowState`，后面扩展时不会推倒重来。

### 3.2 VRS 修正要影响“控制效能”

Leishman 强调 VRS 会带来桨叶拍振、旋翼控制能力损失；如果发生在尾桨上，还会严重影响偏航控制。这提示本项目的 VRS 修正不应只写成：

$$
T_{\mathrm{eff}}=k_TT
$$

还应预留操纵效能修正：

$$
\frac{\partial T_{\mathrm{eff}}}{\partial \theta_0},
\quad
\frac{\partial L_{\mathrm{eff}}}{\partial \theta_{1c}},
\quad
\frac{\partial M_{\mathrm{eff}}}{\partial \theta_{1s}}
$$

也就是说，进入 VRS 后不仅“升力变小”，而且“加总距/打周期变距的效果变差且更乱”。

### 3.3 VRS 和自转不能混为一谈

Leishman 把 VRS 与风车制动/自转状态分开处理。对本项目来说，VRS 判据必须包含“有功率输入”或“非自转”条件，否则正常自转下降可能被误判为 VRS。

建议阶段二判据至少区分：

| 状态 | 功率/总距 | 下降率 | 旋翼流态 |
|---|---|---|---|
| 正常下降 | 有功率 | 小到中等 | 正常工作 |
| VRS | 有功率 | 中等到较大 | 涡环/湍流尾迹 |
| 自转 | 低功率或无功率 | 可较大 | 风车制动 |

## 4. 和 Padfield 的对照

Padfield 的重点是仿真建模：VRS 边界可用标志位，但区域内部必须经验修正。Leishman 的重点是旋翼气动图像：为什么这个区域不能用简单动量理论连续外推。

两者合起来，给本项目一个清楚结论：

```text
动量理论
  -> 可用于正常工作/风车制动两端
  -> 不能单独覆盖 VRS 内部

VRS 模块
  -> 入口边界: 用下降率、水平速度、诱导速度比、功率输入判别
  -> 区域内部: 用经验诱导速度/拉力/操纵效能修正
```

## 5. 碰撞问题

这段阅读逼出的关键问题是：

> 我们的 VRS 模型是只要“掉高度更快”就算成功，还是要表现“控制效能也变差”？

阶段一结论应选择后者，但第一版实现可以分两步：

1. 先实现有效拉力下降和诱导速度异常。
2. 再加入总距/周期变距控制效能退化和非定常扰动。
