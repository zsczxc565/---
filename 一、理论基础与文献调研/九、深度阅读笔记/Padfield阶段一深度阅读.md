# Padfield 阶段一深度阅读笔记

来源：G. D. Padfield, *Helicopter Flight Dynamics: The Theory and Application of Flying Qualities and Simulation Modelling*。

本轮阅读目标不是通读全书，而是服务阶段一重启：先抽出“轴系、六自由度方程、仿真建模层级、入流/VRS、验证思想”这几根骨架。

## 1. 一句话摘要

Padfield 的核心启发是：直升机仿真模型不是越复杂越好，而是要围绕任务需求选择足够的物理保真度，并用可验证的六自由度框架把旋翼、尾桨、机身、尾翼和入流效应装配起来。

## 2. 全局结构地图

```text
建模目的
  -> 选择模型保真度
    -> Level 1: 六自由度机体 + 准定常旋翼/低阶入流
    -> Level 2/3: 叶素、弹性桨叶、非定常尾迹、相互干扰
  -> 冻结轴系和状态量
    -> 机体系速度: u, v, w
    -> 机体系角速度: p, q, r
    -> 欧拉角: phi, theta, psi
  -> 建立六自由度刚体方程
    -> 合外力: X, Y, Z
    -> 合外矩: L, M, N
  -> 建立部件气动力/力矩
    -> 主旋翼
    -> 尾桨
    -> 机身
    -> 平尾/垂尾
  -> 处理旋翼入流和 VRS
    -> 正常: 动量理论/动态入流
    -> VRS: 边界可判别，区域内部需经验修正
  -> 验证
    -> 功能保真度
    -> 物理保真度
```

## 4. 轴系阅读

Padfield 的机体系采用英文飞行动力学中常见的下正轴系：

| 量 | Padfield 定义 | 项目处理 |
|---|---|---|
| $x_b$ | 向机头前方 | 作为待转换源 |
| $y_b$ | 向右舷 | 作为待转换源 |
| $z_b$ | 向下 | 与中俄轴系逐项核验 |
| $u,v,w$ | 沿 $x_b,y_b,z_b$ 的速度分量 | 不直接作为项目最终符号 |
| $p,q,r$ | 绕 $x_b,y_b,z_b$ 的角速度分量 | 不直接作为项目最终符号 |
| $X,Y,Z$ | 沿 $x_b,y_b,z_b$ 的外力 | 阶段二需写转换矩阵 |
| $L,M,N$ | 绕 $x_b,y_b,z_b$ 的外矩 | 阶段二需写力矩正方向 |

这一点很重要：Padfield 的方程适合做英文源参考，但不能直接变成项目方程。你要求“中俄轴系”，所以阶段一 P0 必须先冻结项目轴系，再把 Padfield、Leishman、Prouty、Johnson 的公式逐项换轴。

## 5. 六自由度方程骨架

在 Padfield 轴系下，机体平动方程可整理为：

$$
\dot{u}=rv-qw+\frac{X}{m}-g\sin\theta
$$

$$
\dot{v}=pw-ru+\frac{Y}{m}+g\cos\theta\sin\phi
$$

$$
\dot{w}=qu-pv+\frac{Z}{m}+g\cos\theta\cos\phi
$$

转动方程保留惯性积 $I_{xz}$：

$$
I_{xx}\dot{p}=(I_{yy}-I_{zz})qr+I_{xz}(\dot{r}+pq)+L
$$

$$
I_{yy}\dot{q}=(I_{zz}-I_{xx})rp+I_{xz}(r^2-p^2)+M
$$

$$
I_{zz}\dot{r}=(I_{xx}-I_{yy})pq+I_{xz}(\dot{p}-qr)+N
$$

欧拉角与机体系角速度关系为：

$$
p=\dot{\phi}-\dot{\psi}\sin\theta
$$

$$
q=\dot{\theta}\cos\phi+\dot{\psi}\sin\phi\cos\theta
$$

$$
r=-\dot{\theta}\sin\phi+\dot{\psi}\cos\phi\cos\theta
$$

阶段二不能直接复制这组式子，而要先完成“Padfield 轴系 $\rightarrow$ 中俄轴系”的变换。

## 6. 建模保真度判断

Padfield 把建模复杂度和应用目标绑在一起。对本项目，第一版适合采用 Level 1 思路：

| 模块 | 第一版建议 | 原因 |
|---|---|---|
| 机体 | 非线性六自由度刚体 | 是所有部件外力/外矩的装配骨架 |
| 主旋翼 | 准定常旋翼 + 低阶入流 | 足够支撑悬停、下降、VRS 判别原型 |
| 入流 | 均匀入流起步，预留动态入流接口 | VRS 首先影响诱导速度 |
| 挥舞 | 准定常等效桨盘倾斜 | 避免第一版被桨叶弹性自由度拖住 |
| VRS | 判据 + 经验修正 | 简单动量理论不能描述 VRS 区域内部 |
| 验证 | 功能保真度优先 | 先看进入、失速样下降、改出趋势是否可信 |

这不是降低目标，而是把模型复杂度压到“能解释、能验证、能迭代”的位置。

## 7. VRS 阅读结论

Padfield 对 VRS 的态度很清楚：

- VRS 出现在近垂直下降、下降率中等到较大、旋翼尾迹与上洗缠绕的区域。
- 进入 VRS 后，旋翼有效升力会显著下降，继续加总距可能进一步降低效率。
- 简单动量理论能解释正常爬升、悬停、风车制动等区域，但不能真实描述 VRS 内部强非定常涡结构。
- 对仿真模型而言，VRS 边界可以做标志位；但进入区域后的垂向力、诱导速度和操纵效能必须引入经验修正。

这直接支持本项目的 VRS 模块分成两层：

```text
VRS_Detection
  -> 判断是否接近/进入 VRS 区域

VRS_Correction
  -> 修正诱导速度
  -> 修正有效拉力
  -> 可选加入非定常扰动
```

## 8. 动态入流阅读结论

Padfield 给出的动态入流骨架可以抽象为三个入流状态：

$$
\lambda_0,\quad \lambda_{1s},\quad \lambda_{1c}
$$

其中 $\lambda_0$ 表示均匀入流，$\lambda_{1s}$、$\lambda_{1c}$ 表示一阶谐波入流。它们可由推力、滚转气动力矩和俯仰气动力矩驱动。

阶段二建议：

1. 初版只实现 $\lambda_0$，确保悬停和垂直下降正确。
2. 接口上预留 $\lambda_{1s}$、$\lambda_{1c}$，后续用于处理桨盘非均匀入流和滚转/俯仰耦合。
3. VRS 修正优先作用于 $\lambda_0$ 和有效拉力，再逐步扩展到一阶谐波扰动。

## 9. 碰撞问题

这轮阅读真正撞出来的问题是：

> 本项目到底要先做“物理上尽量完整的 VRS 流场模型”，还是先做“飞行动力学上可验证的 VRS 进入-恶化-改出模型”？

Padfield 的答案偏向后者。对阶段一二来说，也应该先选择后者：先把轴系、六自由度、旋翼入流、判据和验证场景做稳，再逐步提高旋翼/尾迹保真度。

## 11. 下一步线索

下一轮深读优先做三件事：

1. 找到课程资料或中文/俄式教材中的中俄轴系原始定义，冻结项目轴向。
2. 对 Leishman、Johnson 扫描版做 OCR 或人工页图阅读，补旋翼动量理论、叶素理论和 VRS/下降飞行依据。
3. 用 Prouty 补配平、操纵量方向和工程参数估算，尤其是总距、周期变距、尾桨总距的方向约定。
