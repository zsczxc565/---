# 论文笔记：动态入流模型与本项目取舍

## 来源

Dynamic inflow model comparison thesis:

<https://open.metu.edu.tr/bitstream/handle/11511/25959/index.pdf>

## 一句话

动态入流模型决定旋翼诱导速度如何随飞行状态、操纵和桨盘位置变化，是连接叶素气动、挥舞响应和 VRS 修正的中间层。

## 资料中的主要模型

| 模型 | 特点 | 对本项目的适用性 |
|---|---|---|
| 均匀入流 | 只考虑桨盘平均诱导速度，简单稳定 | 适合第一版模型 |
| Payne 线性入流 | 在均匀入流基础上加入线性分布 | 可作为第一版升级 |
| Pitt-Peters | 低阶动态入流，常用于飞行动力学仿真 | 适合第二版 |
| Peters-He | 广义动态尾流，可用更多状态描述高阶入流 | 适合后续高保真扩展 |

## 对本项目的判断

本项目目标是“涡环状态建模与仿真”，不是单独研究动态入流。因此初版应避免被高阶动态入流拖住。推荐路线：

```text
均匀入流
  -> 加 VRS 经验修正
  -> 跑通进入/改出场景
  -> 如响应过于生硬，再升级 Pitt-Peters
```

## 与叶素理论的关系

叶素理论需要局部入流角和局部迎角，因此诱导速度模型会直接影响：

- 旋翼拉力
- 旋翼反扭矩
- 桨盘载荷
- 挥舞角估计
- VRS 中的拉力损失

## 与 VRS 的关系

VRS 的核心异常之一就是诱导速度不再按正常动量理论分支变化。动态入流模型本身不等于 VRS 模型，但它提供了一个很自然的接口：

```text
正常入流模型输出
  -> VRS 判据判断
  -> VRS 修正诱导速度
  -> 叶素/旋翼力计算
```

## 阶段二建议

| 版本 | 入流选择 | 理由 |
|---|---|---|
| V0 | 均匀入流 | 最快跑通模型 |
| V1 | 均匀入流 + VRS 修正 | 能表现涡环进入/改出 |
| V2 | Pitt-Peters + VRS 修正 | 改善动态响应和桨盘分布 |
| V3 | Peters-He 或自由尾迹 | 仅在时间允许、需要高保真时考虑 |
