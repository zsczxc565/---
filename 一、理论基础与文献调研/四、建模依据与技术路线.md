# 四、建模依据与技术路线

## 1. 建模对象

本项目建模对象建议限定为常规单主旋翼带尾桨直升机。理由是课程资料中关于操纵、尾桨、配平、稳定性的大部分内容都以该构型为基础，且 VRS 对主旋翼垂向诱导速度和拉力的影响最直接。

模型应覆盖：

- 机体刚体六自由度运动
- 主旋翼气动力和力矩
- 尾桨侧向力和反扭矩平衡
- 机身、平尾、垂尾气动力的简化表达
- 旋翼诱导速度
- 涡环状态判别与修正
- 典型进入和改出场景

## 2. 坐标系依据

本项目坐标系采用中俄轴系作为建模基准。坐标轴、姿态角、速度分量、角速度分量、力矩分量和下降率符号已在 [轴系与符号冻结约定](../docs/axis-conventions.md) 中冻结。

课程资料明确使用多坐标系描述直升机运动。阶段二应至少冻结以下坐标系：

| 坐标系 | 用途 |
|---|---|
| 地面坐标系 | 描述位置、高度、航迹和地速 |
| 机体坐标系 | 描述速度分量、角速度、姿态运动和六自由度动力学 |
| 速度坐标系 | 描述气流方向、迎角、侧滑角 |
| 桨轴系/旋翼坐标系 | 计算旋翼力、诱导速度、桨盘倾斜和挥舞 |

关键约束：

- 不直接照搬英美教材中的 NED/右手轴系公式；凡引用英文教材公式，必须做轴系和符号转换说明。
- 阶段二必须给出地面坐标系、机体坐标系、速度坐标系、桨轴系之间的方向定义、转移矩阵和正方向约定。
- 机体坐标系应作为六自由度方程主坐标系。
- 旋翼力应先在旋翼/桨轴系中计算，再转换到机体坐标系。
- 重力应从地面坐标系转换到机体坐标系。
- 高度、垂向速度和下降率应保持符号定义一致，尤其是 VRS 判据中的下降率。

冻结后的轴系对照表：

| 项目 | 中俄轴系定义 | 英文教材常见定义 | 转换处理 |
|---|---|---|---|
| 地面坐标轴 | $X_D$ 为初始航向或预设方向，$Y_D$ 向上，$Z_D$ 按右手系确定 | 常见为 NED 或 ENU | 项目高度 $H=Y_D$，下降率 $V_d=-\dot{H}$ |
| 机体坐标轴 | $X$ 向前，$Y$ 向上，$Z$ 按右手系指向驾驶员右手侧 | 课程第 80-81 页所示西方体轴系 | 英文公式进入项目前先转换到 $OXYZ$ |
| 姿态角 | $\gamma$ 右侧倾为正，$\vartheta$ 抬头为正，$\psi$ 左偏航为正 | 常见为 $\phi_B$、$\theta_B$、$\psi_B$ | $\phi_B=\gamma$，$\theta_B=\vartheta$，$\psi_B=-\psi$ |
| 速度分量 | $V_x,V_y,V_z$ 分别沿 $X,Y,Z$ 正向 | 常见为 $u_B,v_B,w_B$ | 优先在规格中使用 $V_x,V_y,V_z$ |
| 角速度分量 | $\omega_x,\omega_y,\omega_z$ 分别绕 $X,Y,Z$ 正向 | 常见为 $p,q,r$ | $p=\omega_x$，$q=\omega_z$，$r=-\omega_y$ |
| 力矩分量 | $M_x,M_y,M_z$ 分别绕 $X,Y,Z$ 正向 | 常见为 $L,M,N$ | $L=M_x$，$M=M_z$，$N=-M_y$ |

## 3. 状态量依据

六自由度模型至少需要以下状态量：

| 类别 | 状态 |
|---|---|
| 平动速度 | 机体系纵向、横向、垂向速度 |
| 角速度 | 滚转、俯仰、偏航角速度 |
| 姿态 | 滚转角、俯仰角、偏航角 |
| 位置 | 地面坐标下的位置和高度 |

VRS 建模额外需要：

| 类别 | 变量 |
|---|---|
| 旋翼入流 | 平均诱导速度、轴向入流、前进比 |
| 旋翼载荷 | 主旋翼拉力、反扭矩、桨盘载荷 |
| VRS 指标 | 下降率、水平速度、功率/总距状态、VRS 标志、修正系数 |

## 4. 外力和外矩依据

课程资料强调：直升机运动方程与刚体动力学形式相同，但外力和外矩计算复杂。因此模型结构应分成两层：

1. 刚体动力学层：只负责由合外力和合外矩推进状态。
2. 气动/推进层：负责计算旋翼、尾桨、机身、平尾、垂尾等部件贡献。

推荐外力/外矩来源：

| 部件 | 力/力矩贡献 |
|---|---|
| 主旋翼 | 拉力、后向力、侧向力、反扭矩、桨毂力矩 |
| 尾桨 | 侧向力、反扭矩、偏航力矩 |
| 机身 | 阻力、侧力、升力和相应力矩 |
| 平尾 | 俯仰稳定和下洗影响 |
| 垂尾 | 航向稳定和侧向/偏航阻尼 |
| 重力 | 机体系分量 |

## 5. 旋翼气动依据

阶段一资料支持两条旋翼气动路线：

1. 动量理论：把旋翼看作作用于气流的圆盘，用诱导速度建立拉力关系。
2. 叶素理论：把桨叶分成微段，计算局部速度、迎角、升阻力，再沿桨叶积分。

本课题建议采用“叶素理论为主，动量/入流模型给诱导速度”的组合：

```text
操纵量 + 飞行状态
  -> 叶素局部速度和迎角
  -> 叶素升阻力
  -> 主旋翼拉力/力矩
  -> 动态入流或经验入流修正
  -> VRS 修正
```

原因：

- 课程资料已明确动量理论和叶素理论可结合求解旋翼力。
- VRS 本质上强烈影响诱导速度和旋翼有效拉力。
- 叶素框架便于后续加入总距、周期变距、挥舞和桨盘倾斜。

## 6. 挥舞运动依据

课程资料将挥舞分为三类：

| 类型 | 含义 | 对本项目的用途 |
|---|---|---|
| 吹风挥舞 | 由飞行状态和来流引起，不直接受飞行员控制 | 用于前飞/下降时桨盘姿态变化 |
| 操纵挥舞 | 由周期变距操纵引起 | 用于驾驶杆到俯仰/滚转力矩映射 |
| 随动挥舞 | 由机体俯仰/滚转角速度引起 | 用于运动耦合和阻尼近似 |

阶段二可以先采用准定常挥舞假设：不完整积分桨叶挥舞微分方程，而是把挥舞效果等效为桨盘倾斜和桨毂力矩。若模型稳定后，再提高为一阶动态挥舞。

## 7. 配平依据

课程资料明确：稳定飞行分析的核心是“给定飞行状态，求所需操纵量和姿态”，也就是配平。

本项目至少需要三个基准配平点：

| 配平点 | 目的 |
|---|---|
| 悬停 | 检查重力与旋翼拉力平衡，验证诱导速度和功率量级 |
| 正常垂直下降 | 检查非 VRS 下降段的垂向速度和拉力变化 |
| 低速前飞/陡降 | 作为进入 VRS 的初始条件 |

配平输出应包括：

- 总距
- 纵向周期变距
- 横向周期变距
- 尾桨总距
- 俯仰姿态
- 滚转姿态
- 旋翼诱导速度

## 8. VRS 建模依据

VRS 应作为旋翼入流和有效拉力的异常状态建模，而不是改变刚体动力学基本形式。

推荐结构：

```text
飞行状态 + 主旋翼状态
  -> VRS_Detection
  -> VRS_Correction
  -> 修正诱导速度 / 修正有效拉力 / 加入扰动
  -> 主旋翼力和力矩
```

VRS 判据建议先采用工程经验型：

| 判据 | 含义 |
|---|---|
| 低水平速度 | 低于有效转换升力速度或设定阈值 |
| 有明显下降率 | 垂直或近垂直下降，下降率超过阈值 |
| 仍有功率输入 | 总距/功率不是自转状态 |
| 旋翼处于自身下洗中 | 用下降率与诱导速度比值表示 |

VRS 修正建议先从平均入流修正开始：

- 正常工况：使用动量理论/动态入流。
- VRS 边界附近：对诱导速度加入经验修正。
- 深度 VRS：降低有效拉力，加入非定常波动或扰动项。

王畅等 2023 年风洞试验进一步提示，VRS 检测不应只输出单一开关，还应区分：

1. 拉力阻尼由正转负的进入边界。
2. VRS 内部的拉力损失和功率沉陷区域。
3. 拉力阻尼恢复但拉力性能尚未恢复的过渡区。
4. 拉力恢复至悬停等值水平的退出边界。

阶段二可将该论文的半经验边界与 NASA Johnson 边界并列实现为可切换候选，不在缺少目标旋翼数据时直接宣称其中任一边界为最终模型。详细依据见[王畅等 2023 年涡环边界风洞试验笔记](八、专题文献笔记/王畅等2023年涡环边界风洞试验笔记.md)。

Johnson 路线的正式升级顺序建议为：

```text
普通动量入流
  -> 消除理想自转奇异性的基线入流
  -> D-N-X-E 三段 VRS 入流增量
  -> 一阶入流滞后
  -> 可选低频扰动
```

其中准定常入流写为：

$$
v_{\mathrm{QS}}
=\kappa\left(v_{\mathrm{base}}+f\Delta v_{\mathrm{VRS}}\right)
$$

完整标称参数和连接条件见[Wayne Johnson 涡环状态模型笔记](八、专题文献笔记/Wayne%20Johnson涡环状态模型笔记.md)。

## 9. 阶段二建模选择建议

| 模块 | 阶段二建议 |
|---|---|
| 六自由度方程 | 使用非线性刚体方程 |
| 主旋翼 | 叶素 + 均匀入流起步 |
| 动态入流 | 初版用一阶平均入流，后续升级 Pitt-Peters |
| 挥舞 | 初版用准定常等效桨盘倾斜 |
| VRS | 初版用判据 + 诱导速度/拉力修正 |
| 配平 | 先做悬停和定常下降 |
| 验证 | 用 FAA 判据、NASA VRS 响应和课程配平规律做交叉检查 |
